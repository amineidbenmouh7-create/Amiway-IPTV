# Amiway-IPTV
"Automated Pinterest marketing app for my IPTV business. Creates and publishes 5 pins daily promoting live TV channels, movies, series, and sports packages. Each pin includes AI-generated titles/descriptions and custom images with a WhatsApp contact link. Personal use only

# Privacy Policy for IPTV Pin Bot

Last updated: June 2026

This application is for personal use only. It does not collect, store, or share any user data.

## Data Collection
This app does not collect any personal data from users. It only accesses your Pinterest account to create and publish pins as authorized by you.

## Third-Party Services
This app uses the Pinterest API and Google Gemini API.

## Contact
For questions, contact via the WhatsApp link provided in the pins.
